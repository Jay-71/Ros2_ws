## 🛠️ Prerequisites

Before running these packages, ensure you have the following installed:
* **ROS2 Distribution** (e.g., Humble, Foxy, or Jazzy)
* **Python 3.x**
* **Gazebo Simulator**
* **Colcon** (Build tool)

## 🚀 Getting Started

Follow these steps to set up the workspace on your local machine.

### 1. Clone the Repository
Open your terminal and clone this repo into your `src` directory (or creates a new workspace):

```bash
mkdir -p ~/ros2_ws/src
cd ~/ros2_ws/src
git clone [https://github.com/Jay-71/Ros2_ws.git](https://github.com/Jay-71/Ros2_ws.git) .
